# animation-ui-kit

[![Deploy to GitHub Pages](https://github.com/theMaxPo/animation-ui-kit/actions/workflows/static.yml/badge.svg)](https://themaxpo.github.io/animation-ui-kit/)

https://themaxpo.github.io/animation-ui-kit/

Этот проект создан как демонстрация навыков frontend-разработки. Проект написан с нуля без использования UI-фреймворков и библиотек компонентов.

## Технологический стек

* **Базовые технологии:** `HTML5`, `CSS3`.
* **Стилизация:** `SCSS` (`SASS`) по методологии **BEM**, `PostCSS`.
* **Скрипты:** `TypeScript` (`ES6+`).
* **Сборка:** `Webpack 5`.
* **Анимации:** `CSS`, `JS` через `requestAnimationFrame`, `Canvas API`, `Lottie`.
* **Однообразие:** `ESLint`, `Prettier`.
* **CI/CD:** сборка и деплой на `GitHub Pages` через `GitHub Actions`.

## Локальный запуск

Для работы с проектом требуется `Node.js`:

```bash
# 1. Клонируем репозиторий
git clone https://github.com/theMaxPo/animation-ui-kit.git
cd animation-ui-kit

# 2. Устанавливаем зависимости проекта
npm install

# 3. Запускаем локальный dev-сервер
npm run dev
```
